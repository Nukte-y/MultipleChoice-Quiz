# MultipleChoice-Quiz
 A quiz with a set time limit.
 
 ## Description

Welcome to my interactive Quiz Application! Test your knowledge with a dynamic multiple-choice quiz featuring questions on various topics.

You can access the quiz page at nukte-y.github.io/MultipleChoice-Quiz/.

## Usage

1. **Getting Started:**
    - Click the "Start Quiz" button to begin the quiz.

2. **Answering Questions:**
    - Respond to the multiple-choice questions.
    - As you select an answer, the next question will appear automatically.
